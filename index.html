Setup Guide

This dashboard can pull live Claude API usage data from a Cloudflare Worker that securely queries Anthropic's Usage API on your behalf. Here's the architecture:

Phone PWA → Your Cloudflare Worker → Anthropic Usage API ↑ Admin API key stored as Worker secret

Steps:

  1. Generate an Admin API key in the Anthropic Console (Settings → API Keys → Admin)
  2. Create a new Cloudflare Worker (e.g. anthropic-usage-proxy)
  3. Store the admin key as a Worker secret: ANTHROPIC_ADMIN_KEY
  4. Have the Worker expose GET /usage returning today's cost and token totals
  5. Paste the Worker URL below

Expected response shape:

{ "date": "2026-04-21", "tokens_in": 12345, "tokens_out": 4321, "cost_usd": 0.12, "month_cost_usd": 4.56, "credits_remaining_usd": 95.44 }
